# Raspberry Pi NAS

Serveur NAS (Network Attached Storage) construit sur Raspberry Pi, accessible sur le réseau local.

## Présentation

Mise en place d'un serveur de fichiers personnel sur Raspberry Pi. Le dossier `screenshots/` contient des captures de l'interface en fonctionnement : configuration réseau, interface d'administration et accès aux fichiers.

## Stack technique

- Raspberry Pi, Linux
- Système de fichiers réseau (NAS)

## Structure

- `screenshots/` — captures du système en fonctionnement
